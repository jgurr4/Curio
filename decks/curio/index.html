Curio

A curated internet layer, not a browser.

No browsing. No scrolling. Only intentional access.

Problem

  • Search apps and feeds are optimized for attention, not intent.
  • Simple tasks get mixed with ads, thumbnails, clickbait, and distractions.
  • Suggested content can be irrelevant or spiritually unhealthy.

What Curio Is

  • Terminal-based, browserless content discovery software.
  • Text-only results for calm, practical decision-making.
  • AI curation plus hard-code filters to remove inappropriate content.
  • User-defined wholesomeness level to tighten output quality.

Core Rules

  • Curio is not a browser.
  • There is no exploration, only intentional access.
  • All outputs are curated before they are shown.
  • Browser unlock is optional, temporary, and allow-list only.

How Discovery Works

  1. User runs `discover` with explicit tags.
  2. Engine gathers candidates from search + trusted databases.
  3. Hard filters remove obvious unsafe/irrelevant matches.
  4. AI curator ranks, rewrites, and reduces to a concise final set.
  5. Results include relevance score and source trust signals.

CLI Commands

discover movies --tags "civil war, inspirational, created in 1949"
discover music --tags "jazz, classic, popular (more than 20,000 records sold), made in 1930"
discover shopping --tags "hammock, 12 feet or longer, comfortable, outdoors, good reviews"

Use Cases

  • Physical products and local places using search + map sources.
  • General research and learning across trusted knowledge sources.
  • Entertainment discovery across movies, music, books, podcasts, games, and software.
  • Future integration with decentralized social updates from trusted people.

Safety and Trust

  • Hard-coded prompt guardrails for curation behavior.
  • User wholesomeness tier controls strictness.
  • Blacklist + user feedback loops for ongoing filtering.
  • Source trust score reflects reliability + model confidence.
  • Web blocker allows only AI-approved domains for limited time.

Business Position

  • Curio is a new access layer for the internet, not a replacement tab.
  • It targets users who want peace, relevance, and productivity.
  • It creates a defensible trust and safety experience around intent-first search.

Roadmap

  • Phase 1: discover engine + filtering + AI curation + trust score.
  • Phase 2: browser blocker and temporary destination controls.
  • Phase 3: broader source adapters and preference learning.
  • Phase 4: trusted social-layer integrations.

Closing

Intentional internet use at command-line speed.

Less noise. Less temptation. Better decisions.
